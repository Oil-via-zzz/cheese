# Wolf Hunter FPS

A simple first-person shooter game built with JavaScript and Three.js.

## How to Play

1. Open `index.html` in your web browser.
2. Click anywhere on the screen to lock the mouse pointer for camera control.
3. Use WASD keys to move around.
4. Click to shoot at wolves.
5. Walk over dropped items to collect them.

## Features

- First-person 3D perspective
- Mouse look controls
- WASD movement
- Shoot wolves to kill them
- Collect wolf paws, heads, and pelts
- Simple inventory system
- Health and ammo display

## Controls

- **WASD**: Move
- **Mouse**: Look around
- **Click**: Shoot (when pointer is locked)

## Requirements

- Modern web browser with WebGL support
- JavaScript enabled

## Files

- `index.html`: Main HTML file
- `style.css`: Styling
- `game.js`: Game logic

Enjoy hunting wolves!
